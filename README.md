# Serverless React + Graphql

A simple solution with React + Redux on client side and a Graphql + Apollo on server side 

## Quick Overview

Client
``` sh
cd client
npm install
npm start
```

Server
``` sh
cd server
npm install
npm install -g serverless
npm start
```

## Deploy 

Client 
``` sh
cd client
npm install
npm build
npm deploy
```

Server
``` sh
cd server
npm install
serverless deploy
```
